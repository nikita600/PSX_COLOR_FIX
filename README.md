# PSX_COLOR_FIX
Color fix mod for PlayStation console based on CXA1645P encoder.

Based on schematic from "Improved Video Board for Yamaha MSXs": https://github.com/RBSC/YamahaVideoBoard

Installation notes
- cut trace as shown on image below
- solder wires as shown on images below
- for NTSC console solder 4.43MHz quartz and solder jumpers J1 2-3 and J2 1-2
- for PAL console solder 3.57MHZ quartz and solder jumpers J1 1-2 and J2 2-3 (shown on images below)
